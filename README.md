# Overview
The Global Barrier Project is a collaborative initiative aiming to better understand the effect of human infrastructures (e.g. settlements, roads, railways) on terrestrial mammalian movement. 

## Core team members: 
Nilanjan Chatterjee, Thomas Mueller (Senckenberg Biodiversity and Climate Research Institute, Frankfurt, Germany); 
Wenjing Xu (University of Massachusetts, Amherst, USA);
Hanna Wiggen-Oosterhoff, Marlee Tucker (Radboud University, Netherlands)
